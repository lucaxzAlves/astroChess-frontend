# Astro Chess Move Quality Assets

Original SVG assets for board move-quality overlays.

## Included
- Brilliant
- Great
- Best
- Excellent
- Good
- Book
- Inaccuracy
- Mistake
- Missed Chance
- Blunder

## Suggested usage
Place the SVG files in:

`public/assets/move-quality/`

Then use `move-quality.css` as a reference for CSS classes.

These assets are original and designed with a cosmic purple Astro Chess identity. They use the same broad move-quality concepts common in chess analysis interfaces, but are not copies of Chess.com assets.
